== Description ==
LodinPay payment gateway for WooCommerce.

== Installation ==
1. Upload the plugin files to the /wp-content/plugins/lodinpay directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to WooCommerce > Settings > Payments.
4. Enable LodinPay and configure your API credentials.
