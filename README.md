# LodinPay WooCommerce Payment Gateway

LodinPay is a WooCommerce payment gateway that enables instant RTP payments.

## Features

- Instant payment processing
- Secure API integration
- WooCommerce native integration

## Installation

1. Upload plugin to WordPress
2. Activate plugin
3. Go to WooCommerce > Settings > Payments
4. Configure Client ID and Secret

## Testing

1. Add product to cart
2. Proceed to checkout
3. Select LodinPay RTP
4. Place order
